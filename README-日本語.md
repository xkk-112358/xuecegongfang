# 学測工房 利用説明書

## プラットフォーム紹介

学測工房はオンラインで問題演習や自己学習、問題集の管理・共有ができる便利なプラットフォームです。難しい操作は必要なく、各種文書形式の問題集をアップロードして問題を自動整理できます。オンライン解答や自動採点機能に対応し、問題集を簡単に共有することも可能です。日常学習、知識の確認、グループでの学習に適しており、初心者でもすぐに使いこなせます。

## 主な機能

### 複数形式の問題集アップロード

Word などの文書ファイルをアップロードでき、手作業で問題を入力する必要がありません。整理済みの練習問題を直接送信するだけで、手間と時間を削減できます。

### 問題自動抽出機能

ファイルをアップロードすると、システムが問題文、選択肢、解答を自動的に判別して分類します。一括インポートが完了したら、すぐに学習を始められます。

### 手軽な問題集共有

自分が用意した問題集を友達やクラスメイトに簡単に共有できます。相手は再度アップロードする必要がなく、そのまま問題演習を行えます。

### 問題集管理機能

アップロード済みのすべての問題集を一覧で確認でき、内容を簡単に閲覧可能です。個人用の学習資料を整理し、分類別に学習を進められます。

## 利用手順

1.プラットフォームにアクセス
ブラウザから公式ホームページを開き、画面の案内に従って操作してください。
2.問題集をアップロード
ホーム画面のアップロードボタンをタップ
パソコン内の問題文書を選択して送信
自動抽出が完了したら準備完了です
3.問題集を共有
共有機能を開く
対象の問題集を選び、共有リンクまたは認証コードを作成
他人に送信することで、誰でも利用できるようになります

## 利用上の注意事項

アップロードする問題はレイアウトを整え、解答を明確に記載してください。共有作業を行う際はプラットフォームを正常に起動させておいてください。解答中に画面を閉じると学習データが消失する恐れがあります。本サービスは学習目的のみに使用し、違反コンテンツの投稿を禁止します。操作がシンプルなので、初めての方も安心して利用できます。
