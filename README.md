# 学测工坊 使用说明

## 平台简介
学测工坊是一款便捷实用的在线刷题自测、题库管理共享平台，无需复杂操作，支持上传各类文档题库，自动拆解题目，快速在线答题、自动判分，还能轻松分享题库给他人，适合日常学习、知识点自测、小组共用刷题，操作简单，上手即用。

## 核心功能介绍

### 多格式题库上传

支持上传 Word、文档类题库文件，不用手动录入题目，直接上传整理好的习题，省去逐题输入的麻烦，省时省力。

### 题目自动提取

上传题库后，平台自动识别、拆分题目、选项、答案，无需手动编辑整理，一键完成题库导入，即刻生成可答题题库。

### 题库便捷分享

将自己导入的题库，直接分享给同学、好友，他人可直接使用你分享的题库答题，无需重复上传，实现题库共享共用。

### 题库管理

可查看平台内所有已导入的完整题库，清晰浏览所有题目内容，管理个人学习习题，分类刷题更方便。

### 简易使用步骤

第一步：进入平台
打开浏览器，进入学测工坊首页，界面简洁清晰，直接点击对应功能按钮操作即可。
第二步：上传题库
    点击首页上传题库按钮
    选择本地整理好的习题文档（选择题、填空题均可）
    确认上传，等待平台自动提取题目，提示上传成功即可
第三步：分享题库
    点击题库分享
    选择已上传的题库，生成分享链接 / 分享口令
    发送给需要共享的好友，对方打开即可直接使用该题库刷题
## 使用注意事项

    上传的题库，题目排版清晰、答案标注明确，不要有杂乱格式、多余空格，平台能更精准识别题目
    分享题库时，保持平台正常打开，确保好友能正常访问使用
    答题过程中不要随意关闭页面，避免答题数据丢失
    题库仅用于学习自用，请勿上传违规、侵权类习题内容
    操作简单无复杂步骤，全程一键式操作，新手可直接使用
